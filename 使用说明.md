# 网络控制器 - 使用说明

## 快速开始

### 方式一：一键打包安装（推荐）

双击运行 `一键打包安装.bat`，程序会自动完成打包和安装。

### 方式二：手动操作

1. 打包程序：
```bash
python build.py
```

2. 安装程序：
```bash
python installer.py
```

## 安装后

程序会被安装到：`C:\Program Files\NetworkController\`

- ✅ 自动设置开机启动
- ✅ 后台隐藏运行（无窗口）
- ✅ 通过浏览器访问控制

## 访问控制面板

### 本机访问
打开浏览器访问：`http://localhost:5000`

### 局域网访问
1. 查看本机IP：运行 `ipconfig` 命令
2. 其他设备访问：`http://<本机IP>:5000`

## 控制选项

- **正常模式 (0%)** - 网络正常运行
- **中等延迟 (50%)** - 50%丢包率（需要额外工具）
- **完全阻断 (100%)** - 完全断网

## 卸载程序

运行：`C:\Program Files\NetworkController\uninstall.bat`

## 注意事项

⚠️ **重要提示**：
- 程序需要管理员权限才能修改网络设置
- 首次运行可能需要允许防火墙访问
- 100%阻断会完全断开网络连接
- 确保防火墙允许5000端口访问

## 技术细节

- 打包工具：PyInstaller
- 运行方式：后台服务（无控制台窗口）
- 自启动方式：注册表 Run 键
- 端口：5000
- 权限：需要管理员权限修改网络

## 故障排除

### 无法访问控制面板
1. 检查程序是否运行：任务管理器查找 `NetworkController.exe`
2. 检查防火墙设置
3. 尝试重启程序

### 无法修改网络设置
- 确保以管理员权限运行
- Windows: 右键exe选择"以管理员身份运行"

### 局域网无法访问
1. 检查防火墙是否允许5000端口
2. 确保设备在同一局域网
3. 检查本机IP地址是否正确
