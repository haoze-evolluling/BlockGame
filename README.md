# 网络控制器

用于调节电脑网络丢包率的工具，支持两档控制：0%、100%。

## 功能特性

- 🎮 两档丢包率控制（正常/完全阻断）
- 🌐 支持局域网访问
- 🎨 现代炫酷的Web界面
- 👻 后台隐藏运行（无窗口）
- 🚀 开机自动启动

## 使用方法

通过浏览器访问控制面板：

### 本机访问
`http://localhost:5000`

### 局域网访问
`http://<本机IP>:5000`

查看本机IP：运行 `ipconfig` 命令

## 开发模式

如需开发调试，可直接运行源码：

```bash
# 安装依赖
pip install -r requirements.txt

# 运行服务
python backend/server.py
```

## 技术说明

- 后端：Flask + Python
- 前端：原生HTML/CSS/JavaScript
- 网络控制：PowerShell (Windows) / tc (Linux)

## 注意事项

⚠️ **重要提示**：
- 程序需要管理员权限才能修改网络设置
- 100%阻断会完全断开网络连接（30秒后自动恢复）
- 确保防火墙允许5000端口访问
